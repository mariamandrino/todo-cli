# Todo CLI

Una aplicación de lista de tareas que funciona desde la terminal, escrita en Python.

## ¿Qué hace?

- Añadir tareas
- Ver todas las tareas en una tabla con colores
- Marcar tareas como completadas
- Eliminar tareas
- Guardar las tareas automáticamente en un archivo JSON

## Requisitos

- Python 3.x
- rich

## Instalación

Clona el repositorio:

    git clone https://github.com/mariamandrino/todo-cli.git
    cd todo-cli

Instala las dependencias:

    pip install -r requirements.txt

Ejecuta el programa:

    python todo.py

## Tecnologías

- Python 3
- [rich](https://github.com/Textualize/rich) para el formato de la terminal
- Git y GitHub para control de versiones
